# wttr.sh
Get your weather using a simple bash file using wttr.in

## One line version (.\*shrc, ...)
curl: `curl -s "wttr.in/$(curl -s ipinfo.io/city)?Q0"`
<br>wget: `wget -qO- "wttr.in/$(wget -qO- ipinfo.io/city)?Q0"`
